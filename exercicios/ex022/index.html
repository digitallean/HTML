Espaçamentos de texto

Os parágrafos a seguir vão ter formatações de espaçamento diferentes usando as propriedades line-height, letter-spacing e word-spacing. Não se esqueça que é possível misturar todas essas configurações em um único seletor. Só fiz tudo separado para você conseguir entender melhor o resultado visual de cada um deles.
